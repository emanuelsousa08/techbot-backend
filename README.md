# ⚙️ TechBot Backend

## 📌 Descrição

Backend responsável pela autenticação de usuários, gerenciamento de conversas e integração com a API de IA generativa.

---

## 🛠️ Tecnologias

* Node.js
* Express
* Sequelize
* SQLite
* JWT (autenticação)

---

## 🔗 Endpoint

https://techbot-backend-llh2.onrender.com

---

## 🔐 Variáveis de Ambiente

```env
JWT_SECRET=your_secret
GOOGLE_API_KEY=your_api_key
```

---

## ▶️ Como rodar localmente

```bash
npm install
node index.js
```

---

## 📡 Principais rotas

* `/auth` → autenticação
* `/chats` → conversas
* `/api/gemini/generate-content` → integração com IA
